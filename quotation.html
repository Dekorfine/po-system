📖 报价单使用指南

中文版操作流程 · 帮助你快速生成专业英文报价文档

🎯 这个工具能做什么？
生成 4 种英文商务文档：报价单 (Quotation)、形式发票 (PI)、采购订单 (PO)、装箱单 (Packing List)。支持 12 家公司预设、7 种语言、PDF 打印、CSV 导出，全团队 Supabase 云同步。
🚀 5 分钟上手
  1. 选择卖方公司：在「Order Info」区点选 Vakkerlight / Dekorfine / Radilum 等公司预设（地址电话邮箱自动填入）
  2. 选择文档类型：Quotation（询价报价） / PI（形式发票）/ PO（采购订单）/ Packing List（装箱单）
  3. 填买家信息：客户姓名、邮箱、地址（粘贴一段就行，会自动识别）
  4. 添加产品行：每行填 SKU、规格、单价、数量；点 "+ Add Item" 添加更多
  5. 预览 + 打印：右侧 4 个 Tab 实时预览不同文档 → 点 "Export PDF" 或浏览器打印（Ctrl+P）
💾 保存与协作
  • 自动云保存：点 ☁ 保存到云端 或按 Ctrl+S，全团队 Supabase 同步
  • 智能去重：同一个 PI 编号多次保存只更新最新版，不会生成多个文件
  • 三视图：
    • My Quotes：只看你自己的
    • Team：看全团队所有人的
    • Trash：回收站（30 天内可恢复）
  • 状态标签：Draft 草稿 / Sent 已发 / Won 成单 / Lost 丢单 / Archived 归档
🌐 多语言切换
  • 右上角语言选择：English / Bilingual 中英对照 / Español / Deutsch / Français / العربية / Русский
  • 切换语言后所有标签自动翻译，但产品名/规格保持原内容
🖼️ 产品图片
  • 每行产品都有 "Image" 区域，可以从剪贴板粘贴图片（Ctrl+V）
  • 建议每张图 < 200KB，否则保存会变慢
  • 图片会一同保存到云端，PDF 导出时直接嵌入
⌨️ 快捷键
Ctrl + S保存到云端
Ctrl + P打印（PDF）
Esc关闭弹窗
Tab切换输入字段
💡 小技巧
  • Duplicate as New：在 Cloud Records 列表选中一条 → 点此按钮 → 基于这条新建一份（适合做相似订单）
  • Export CSV：把列表导出 Excel 做分析
  • 状态筛选：列表上方 Draft / Sent / Won 标签可快速筛选
  • 自动 PI 递增：留空 PI Number，保存时会自动按你历史记录递增
⚠️ 注意事项
  • 从「统一工作台」跳转过来会自动登录，无需填写姓名
  • 直接打开 quotation.html 需要先在 ⚙️ 设置里填写姓名
  • 云同步依赖 Supabase 项目，若长时间显示 "Loading"，检查网络/Supabase 状态

📋 怎么用?两种方式 ▾
① 做全新报价 — 直接在下面「主表单」从上往下填:选卖方公司、填客户信息、加产品行,右上选输出语言;填好点最上方「👁 预览 & 导出」看效果并导 PDF。
② 拿别人 / 自己做过的报价来改(更快,推荐)— → 选「👥 团队」就能看到全部同事做过的报价 → 搜客户名 / PI 号 → 点那一条会载入到下面表单 → 改内容 → 点「📋 复制为新报价」保存(不会覆盖原来那张)。
